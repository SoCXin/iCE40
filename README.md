### [描述](https://github.com/sochub/iCE40/wiki) 

[iCE40](https://github.com/sochub/iCE40)  系列是超低功耗非易式FPGA器件，640到7680个逻辑单元和触发器，每个器件包含8到32个RAM区块，每个区块有4Kb存储，用于数据存储和缓冲，特别适合对成本敏感和量大的应用。

### 关键参数：

* 40nm
* 2.5 x 2.5 mm， 0.4mm pitch
